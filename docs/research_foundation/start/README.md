# Overleaf Structure

This folder is organized for direct upload to Overleaf.

## Main Files

- `main.tex`: root dissertation file
- `references.bib`: BibTeX database
- `nottingham-logo.png`: title-page logo

## Subfolders

- `frontmatter/`: title page and abstract
- `chapters/`: one file per thesis chapter
- `appendices/`: appendix stubs

## Recommended Overleaf Workflow

1. Upload the entire `start/` folder to Overleaf.
2. Set `main.tex` as the root document.
3. Keep figures in this folder tree so relative paths keep working.
4. Add citations into `references.bib`.
